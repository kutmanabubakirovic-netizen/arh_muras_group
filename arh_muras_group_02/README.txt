## 📋 ФАЙЛЫ ДЛЯ ЗАГРУЗКИ НА СЕРВЕР

Загрузите эти файлы в один папку (например, `/public` или `/www`):

```
index.html              ← главная страница
projects.html          ← портфолио (с защитой)
team.html              ← команда
style.css              ← все стили (основные + дополнительные)
script.js              ← JavaScript с аутентификацией
animations.js          ← анимации и темный режим
```

**Всё остальное:** HTML, CSS и JS содержат всё необходимое! Дополнительных файлов не требуется.

## 📝 ДАННЫЕ СОХРАНЯЮТСЯ

Все загруженные проекты сохраняются в **localStorage браузера**:
- Если вы закроете браузер — проекты останутся
- Если вы очистите историю браузера — проекты удалятся
- Каждый браузер имеет свою копию проектов

**Совет:** Регулярно делайте резервные копии данных!

## ❌ ЕСЛИ ЧТО-ТО НЕ РАБОТАЕТ

### Проблема: "Кнопка ДОБАВИТЬ ОБЪЕКТ не видна"
**Решение:** Вы не авторизованы. Нажмите "только для сотрудников >" и введите логин/пароль.

### Проблема: "CSS не загружается (сайт выглядит плохо)"
**Решение:** Убедитесь, что файл `style.css` в одной папке с HTML файлами.

### Проблема: "JavaScript не работает"
**Решение:** Убедитесь, что файлы `script.js` и `animations.js` в одной папке.

### Проблема: "Форма контактов не отправляется"
**Решение:** Это нормально! Форма временная (просто показывает сообщение). Для реальной отправки требуется backend (сервер).

## 🔄 КАК ДОБАВИТЬ РЕАЛЬНУЮ ОТПРАВКУ ФОРМ?

Если хотите, чтобы формы действительно отправляли письма на email:

**Вариант 1 — Использовать Formspree (бесплатно):**

1. Перейдите на https://formspree.io
2. Создайте аккаунт
3. Создайте новую форму для вашего домена
4. Скопируйте `form_id`
5. Найдите в `index.html` форму и измените:
```html
<!-- ДО -->
<form id="contactForm" onsubmit="submitForm(event)">

<!-- ПОСЛЕ -->
<form action="https://formspree.io/f/YOUR_FORM_ID" method="POST">
```

Потребуется небольшая дополнительная настройка JavaScript.
