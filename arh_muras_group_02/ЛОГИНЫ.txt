## Как изменить пароли?

1. Откройте **script.js** в текстовом редакторе (Notepad++, VS Code)
2. Найдите строку ~14:
```javascript
const employees = {
    'admin': 'admin2026',
    'architect1': 'archi2026',
    // измените пароли здесь
};
```
3. Сохраните файл
4. Перезагрузите сайт в браузере (Ctrl+F5 для полной перезагрузки)

---

## Как добавить нового сотрудника?

В том же файле script.js, добавьте новую строку:
```javascript
'new_username': 'new_password2026',
```

Пример:
```javascript
const employees = {
    'admin': 'admin2026',
    'architect1': 'archi2026',
    'developer': 'dev2026',  // ← новый сотрудник
};
```

## Проблемы?

**"Кнопка ДОБАВИТЬ ОБЪЕКТ не видна"**
→ Введите логин и пароль еще раз. Нажмите F5 (обновить страницу).

**"Забыл пароль"**
→ Измените в script.js, сохраните, обновите браузер.

**"Проекты не сохраняются"**
→ Убедитесь, что localStorage включен (обычно включен по умолчанию).
