## 🎯 АНИМАЦИИ И ЭФФЕКТЫ

### Переход (Transition)
```css
--tr: all 0.36s cubic-bezier(0.25, 0.46, 0.45, 0.94);
```
- **Используется везде** для плавных переходов
- **Времени:** 360 миллисекунд (приятно, не медленно)
- **Функция:** специальная кривая для "живых" движений

### Тени (Shadows)
```css
--shadow: 0 8px 32px rgba(10, 9, 6, 0.08);
--shadow-hover: 0 16px 48px rgba(212, 165, 116, 0.15);
```
- **Обычная тень** — когда элемент просто лежит
- **Тень при наведении** — когда элемент поднимается

### Сияние (Glow)
```css
filter: drop-shadow(0 0 12px rgba(212, 165, 116, 0.6));
box-shadow: 0 0 20px var(--accent);
```
- Используется для логотипа, кнопок
- Создает "теплое свечение" золота

## 🎨 ИЗМЕНЕНИЕ ДИЗАЙНА ЭЛЕМЕНТОВ

### ДО vs ПОСЛЕ

#### 🔵 КНОПКИ

**БЫЛО:**
```css
.btn-primary {
    background: #c49a52;
    box-shadow: none;
}
.btn-primary:hover {
    background: #a67e3e;
}
```
Результат: плоская, скучная кнопка

**СТАЛО:**
```css
.btn-primary {
    background: linear-gradient(135deg, var(--accent), var(--accent-warm));
    box-shadow: 0 4px 12px rgba(212, 165, 116, 0.2);
}
.btn-primary:hover {
    background: linear-gradient(135deg, var(--accent-dark), var(--accent));
    box-shadow: 0 6px 20px rgba(212, 165, 116, 0.3);
    transform: translateY(-2px);
}
```
Результат: красивая кнопка с градиентом, сияет, приподнимается

---

#### 🏠 КАРТОЧКИ ИНТЕРЬЕРА

**БЫЛО:**
```css
.album-card {
    border: 1px solid var(--border);
    box-shadow: 0 2px 8px rgba(0,0,0,0.04);
}
.album-card:hover {
    border-color: var(--accent);
}
```
Результат: плоская карточка, едва заметное изменение

**СТАЛО:**
```css
.album-card {
    box-shadow: 0 4px 16px rgba(0, 0, 0, 0.08);
    border-radius: 12px;
    border: 1px solid var(--border);
}
.album-card:hover {
    border-color: var(--accent);
    box-shadow: var(--shadow-hover);
    transform: translateY(-8px);
    background: rgba(212, 165, 116, 0.03);
}
```
Результат: красивая карточка с объемом, приподнимается при наведении

---

#### 🏷️ КАТЕГОРИЯ (BADGE)

**БЫЛО:**

#### 📐 ЛИНИИ И РАЗДЕЛИТЕЛИ

#### 🌙 ЛОГОТИП

**БЫЛО:**
```css
.logo-svg {
    color: var(--accent);
    transition: transform 0.4s;
}
.logo:hover .logo-svg {
    transform: rotate(90deg);
}
```
Результат: логотип просто крутится

**СТАЛО:**
```css
.logo-svg {
    color: var(--accent);
    transition: transform 0.4s, filter 0.4s;
    filter: drop-shadow(0 0 8px rgba(212, 165, 116, 0.3));
}
.logo:hover .logo-svg {
    transform: rotate(90deg) scale(1.1);
    filter: drop-shadow(0 0 12px rgba(212, 165, 116, 0.6));
}
```
Результат: логотип крутится, светится и увеличивается

## 🎬 ФУНКЦИОНАЛЬНЫЕ АНИМАЦИИ

### REVEAL (Появление при скролле)
```css
@keyframes fadeInUp {
    from { opacity: 0; transform: translateY(20px); }
    to { opacity: 1; transform: translateY(0); }
}
.reveal {
    animation: fadeInUp 0.8s ease-out forwards;
}
```
Элементы появляются снизу вверх при скролле → выглядит очень красиво!

### HOVER (Наведение)
```css
.album-card:hover {
    transform: translateY(-8px);  /* Поднимается на 8px */
    box-shadow: var(--shadow-hover); /* Тень увеличивается */
}
```
При наведении карточка приподнимается → очень приятно на вид!

### FOCUS (Клик на элемент)
```css
input:focus {
    outline: none;
    border-color: var(--accent);
    box-shadow: 0 0 12px rgba(212, 165, 116, 0.15);
}
```
При клике на поле оно светится золотом → понятно что выбрано!

## 🌚 ТЕМНЫЙ РЕЖИМ

Все цвета автоматически меняются при включении темного режима:

```css
[data-theme="dark"] {
    --accent: #d4a574      /* Золото остается таким же ярким */
    --bg: #0f0d0a          /* Фон становится темным */
    --text: #f5f2ed        /* Текст становится светлым */
    /* ... и так далее */
}
```

Темный режим работает везде:
- На главной странице
- На портфолио
- На странице команды
- В модальных окнах
- В формах

## 📱 АДАПТИВНОСТЬ (Мобильные устройства)

На мобильных устройствах дизайн сохраняется:
- Кнопки остаются красивыми
- Карточки остаются красивыми
- Все цвета работают
- Все анимации работают

Единственное что меняется:
- Размер текста и элементов уменьшается
- Сетка карточек становится в 1 колонну
- Меню переходит в бургер (3 линии)

## 🎯 ПСИХОЛОГИЯ ЦВЕТОВ

### Почему именно эти цвета?

**Теплое золото (#d4a574):**
- 🔥 Ассоциируется с теплом, качеством, дороговизной
- 💎 Люксовый, премиум цвет
- 👑 Королевский цвет (используется в дизайне дворцов)
- 🌅 Естественный (как закат, как золотой час)

**Медь/Терракота (#c86d42):**
- 🌍 Землистый, натуральный цвет
- 🏗️ Ассоциируется с архитектурой и строительством
- 🚨 Привлекает внимание (для опасных кнопок)

**Пудровая Роза (#d4a393):**
- 💕 Нежный, женственный цвет
- 🎨 Выглядит дорого и элегантно
- 🤝 Миролюбивый, не раздражает

**Темно-коричневый (#0a0906):**
- 🎭 Профессиональный, серьезный
- 🌲 Природный, органический
- 💼 Деловой вид
